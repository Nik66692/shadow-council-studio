# Canon

Canonical source files live in `source/`. Published version directories are immutable: add newer versions in new directories, never edit or replace an existing published DOCX.

`source/manifest.json` selects the approved current version. The highest numbered directory is not automatically authoritative, and only Niccolò can approve a new current version.

Normalized and registry folders are currently documented placeholders. Sprint 0 stores metadata only and performs no semantic extraction.
