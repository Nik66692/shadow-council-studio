# Phase 1.5 — Canon Review

Canon Review requires an explicit human choice for entry category and canonical status. Neither value has a default.

Approval may combine several pending drafts from the same import run into one canon entry while preserving every ordered source link. Rejection may process pending drafts from different import runs because it creates no merged canon entry.

Every approval or rejection requires a reviewer and rationale and is stored as an immutable decision. Imported source text, anchors and hashes remain unchanged.
