# ADR-0006 — Italian canon, English engineering language and localization readiness

Status: Accepted

## Context

Canon terminology is Italian; implementation benefits from English conventions.

## Decision

Use English for code/technical docs and Italian for UI/canon terminology, without silent translation.

## Consequences

Localization can be added later without losing canonical terms.

## Alternatives considered

Single-language repository.

## Review triggers

Formal localization phase.
