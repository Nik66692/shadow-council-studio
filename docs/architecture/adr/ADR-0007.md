# ADR-0007 — no cloud, telemetry or AI integration during early development

Status: Accepted

## Context

The project is private and local-first.

## Decision

Exclude cloud, telemetry, accounts, auth, remote DB and AI providers in early development.

## Consequences

No secrets or network dependencies are needed.

## Alternatives considered

Hosted backend and early AI APIs.

## Review triggers

Explicit future decision approves optional external integrations.
