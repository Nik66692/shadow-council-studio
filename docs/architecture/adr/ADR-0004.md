# ADR-0004 — stable entity identifiers and append-only history
Status: Accepted

## Context
Names may change while references must remain stable.
## Decision
Reserve immutable `SC-*` ID families and prefer append-only history for versioned records.
## Consequences
References survive renames.
## Alternatives considered
Human-readable slugs as primary keys.
## Review triggers
Identifier collision or new entity families.
