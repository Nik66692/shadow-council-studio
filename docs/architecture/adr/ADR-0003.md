# ADR-0003 — source hierarchy and immutable canonical inputs

Status: Accepted

## Context

Game-design authority must be explicit.

## Decision

Treat Source of Truth v1.3 as immutable primary material and define ordered authority levels.

## Consequences

Software records metadata but never invents canon.

## Alternatives considered

Ad hoc wiki-first editing.

## Review triggers

Niccolò approves a new Source of Truth version.
