# ADR-0001 — Tauri, React, TypeScript and Vite
Status: Accepted

## Context
A local desktop shell needs Rust integration and a maintainable strict frontend.
## Decision
Use Tauri 2, React, TypeScript strict mode and Vite.
## Consequences
Small shell, typed UI and Rust command boundary.
## Alternatives considered
Electron and native-only Rust UI.
## Review triggers
Tauri incompatibility or UI performance constraints.
